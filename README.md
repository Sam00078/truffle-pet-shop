# 宠物商店

## 简介
* 根据 `truffle` 官方教程完成
* 修改了一些官方教程中的代码错误

## 运行
* 安装依赖 `npm install`
* 部署合约 `truffle migrate`
* 运行代码 `npm run dev`

## 调试
* 如果是使用私链 需要配置 `truffle.js` 参考官方文档 [传送门](http://truffleframework.com/docs/advanced/configuration#networks)
* 如果使用 `truffle devploy` 需要配置一下 `src/js/app.js` 里面的 `App.network`
